DANGERS OF USING PUBLIC-WIFI

In my second of university I wrote a paper about the dangers of using public Wi-Fi in places like coffee shops or airports. It explained how hackers can steal personal information through attacks such as spying, fake Wi-Fi, and malware. A real example showed how weak passwords led to a major security breach. The project also talked about tools like Wireshark to test networks safely. In the end, it focused on ways to stay safe, like using encryption, strong passwords, and being careful online

TECHNIQUES FOR IMPROVING CACHE UTILIZATION

In my second year of university I wrote a paper called Techniques For Improving Cache Utilization. The paper explains how cache memory helps computers work faster by storing important data closer to the CPU. It looks at different ways to make cache more efficient, like hardware support, cache hints, cache replacement policies, and cache bypassing. These methods can speed up computers, save energy, and make systems run smoother, but they also have drawbacks like higher cost and complexity. Overall, the research shows that improving cache use is important for better computer performance and should continue to be studied in the future.
